# Python Quiz Game

A simple command-line quiz game written in Python.

## How to Run

1. Make sure you have Python installed.
2. Run the script: `python quizgame.py`
3. Answer the 5 questions by typing your responses.
4. See your score at the end!

## Questions

The quiz includes questions on:
- Geography
- Astronomy
- Literature
- Chemistry
- Biology

## Features

- Case-insensitive answers
- Score tracking
- Percentage calculation

Enjoy the quiz!
